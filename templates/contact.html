EasyMove Sénégal — Un projet étudiant né d'un besoin réel

Notre mission

EasyMove Sénégal est né d'un constat simple : de nombreux touristes, étudiants étrangers et nouveaux arrivants se trouvent démunis face au système de transport sénégalais. Entre les taxis sans compteur, les cars rapides aux itinéraires complexes et les prix qui varient, il est facile de se perdre ou de se faire arnaquer.

Notre objectif est de centraliser en un seul endroit toutes les informations essentielles pour se déplacer sereinement : types de transport, estimations de prix, trajets courants, carte interactive et phrases utiles en wolof pour communiquer avec les habitants.

Ce site a été réalisé dans le cadre d'un projet de fin d'année. Il repose sur une véritable architecture trois-tiers : une base de données SQL, un serveur web Python (Flask) qui traite les recherches d'itinéraires, et une interface HTML/CSS/JS avec carte interactive.
